### 🔧 Installation
1. Clone the repo:
   ```sh
   git clone https://github.com/Melsun-Enterprises/MDM-Assassin.git
   cd MDM-Assassin
   ```
2. Run the setup:
   ```sh
   chmod +x setup.sh && ./setup.sh   # For Linux/Mac
   setup.bat  # For Windows
   ```
